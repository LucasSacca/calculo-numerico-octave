# Cálculo Numérico em Octave

Repositório com exercícios, implementações e estudos desenvolvidos ao longo da disciplina de **Cálculo Numérico**, utilizando **GNU Octave**.

O objetivo deste repositório é organizar, de forma prática, códigos produzidos em aula e em exercícios, contemplando diferentes métodos numéricos e aplicações computacionais.

## Conteúdo do repositório

Atualmente, o repositório reúne scripts relacionados aos seguintes temas:

- **Otimização**
  - Método de Newton para otimização
  - Programação linear com `linprog`

- **Integração numérica**
  - Método do trapézio

- **Interpolação e regressão**
  - Caso de interpolação linear
  - Caso de regressão com comparação de modelos

- **Métodos numéricos para EDOs**
  - Método de Euler
  - Método de Heun
